一、安装客户端
1. 重命名openvpn-2.2.2-install.ex_，将ex_改exe
2. 双击openvpn-2.2.2-install.exe，请安装openvpn客户端到C盘

二、连接与断开连接
有两种方式：命令行方式(CLI)，与图形界面方式（GUI）
CLI会在任务栏中留下一窗口，连接时不可关闭
GUI不会在任务栏中留下痕迹，会在系统托盘中表现连接状态

1.CLI连接方式：（适用平时证书更新频繁的情况）
找到.ovpn文件，右击，选Start Openvpn on this config file
断开连接时在CLI中敲击“两次”F4即可


2.GUI连接方式：（适用证书基本不更新的情况）
重命名3.GUI.ba_扩展名为bat
双击3.GUI.bat批处理文件，配置完成后以后每次连接都从桌面右下角系统托盘处
找到有两个电脑和一个地球的图标，右击选Connect
断开连接时，点选Disconnect
Password是不用的（俺们用的是证书验证。高级货哦！）


三、注意事项
1.如果是在Win7下，本机直接进行PPPoE拨号上网，VPN可能连接失败。
2.正确的做法是先通过家庭网关（家用猫或者路由器）拨号上网，然后将你的计算机接入家庭网关中使用。
